Moroccan Recipes API Documentation

Categories

GET /api/categories

Get all categories with pagination

Query parameters:

  • limit - Number of items per page (default: 10)
  • offset - Number of items to skip (default: 0)

GET /api/categories/:id

Get a single category by ID

POST /api/categories

Create a new category

Body:

{
    "name": "Tagines",
    "description": "Traditional Moroccan stews"
}

Recipes

GET /api/recipes

Get all recipes with pagination

Query parameters:

  • limit - Number of items per page (default: 10)
  • offset - Number of items to skip (default: 0)

GET /api/recipes/search

Search recipes by title

Query parameters:

  • title - Search term for recipe title

POST /api/recipes

Create a new recipe

Body:

{
    "title": "Chicken Tagine",
    "ingredients": ["chicken", "preserved lemon", "olives"],
    "instructions": "1. Brown chicken...",
    "cooking_time": 60,
    "category_id": 1
}
